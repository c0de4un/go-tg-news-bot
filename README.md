# go-tg-news-bot

Telegram bot for news with clients and users.

## Minimal Software requirements
* Go 1.2
* PostgreSQL

## Minimal Hardware Requirements
* 512 MB RAM
* Any CPU x86
* 8 GB file system
* Ethernet

## Build

```shell
go build ./cmd/bot/bot.go
```

## Dev

To start in debug mode, run:
```shell
go run ./cmd/bot/bot.go
```
